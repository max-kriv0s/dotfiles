# Neovim — Шпаргалка

Модальный редактор — в разных режимах клавиши делают разное.

---

## Режимы

| Режим | Как войти | Индикатор |
|-------|-----------|-----------|
| Normal | `Esc` | -- |
| Insert | `i` | -- INSERT -- |
| Visual | `v` | -- VISUAL -- |
| Command | `:` | : в строке снизу |

---

## Запуск

```bash
nvim file.txt    # открыть файл
nvim .           # открыть файловый браузер
nvim             # просто открыть
```

---

## Основные команды (Normal режим)

### Сохранение и выход

| Команда | Действие |
|---------|----------|
| `:w` | сохранить |
| `:q` | выйти |
| `:wq` | сохранить и выйти |
| `:q!` | выйти без сохранения |
| `Space+w` | сохранить (наш маппинг) |
| `Space+q` | выйти (наш маппинг) |

### Навигация

| Клавиша | Действие |
|---------|----------|
| `h/j/k/l` | влево/вниз/вверх/вправо |
| стрелки | то же самое |
| `gg` | начало файла |
| `G` | конец файла |
| `0` | начало строки |
| `$` | конец строки |
| `Ctrl+D` | прокрутить вниз |
| `Ctrl+U` | прокрутить вверх |
| `/{текст}` | поиск вперёд |
| `n/N` | следующее/предыдущее совпадение |

---

## Insert режим

| Клавиша | Действие |
|---------|----------|
| `i` | вставка перед курсором |
| `a` | вставка после курсора |
| `o` | новая строка ниже |
| `O` | новая строка выше |
| `Esc` | выйти в Normal |
| `Cmd+V` | вставить из системного буфера |

> Выделение в Insert режиме недоступно — для этого используй Visual режим.

---

## Visual режим (выделение и копирование)

| Клавиша | Действие |
|---------|----------|
| `v` | выделение символов |
| `V` | выделение строк |
| `ggVG` | выделить всё |
| `y` | скопировать в системный буфер |
| `d` | вырезать |
| `p` | вставить |

---

## Буфер обмена

`clipboard = "unnamedplus"` настроен — обычные `y` и `p` работают с системным буфером напрямую.
Скопировать и вставить в другое приложение:
v → выделить → y → Cmd+V в другом приложении
Вставить из другого приложения в nvim:
i → Cmd+V

## Полезные команды

| Команда | Действие |
|---------|----------|
| `u` | отменить |
| `Ctrl+R` | повторить |
| `yy` | скопировать строку |
| `dd` | вырезать строку |
