# Tmux — Шпаргалка

Терминальный мультиплексор — несколько терминалов в одном окне,
сессии живут даже после закрытия терминала.

Префикс: `Ctrl+A`
Все команды: сначала префикс, потом клавиша.

---

## Установка плагинов

После первого запуска или добавления плагина в конфиг:

```bash
# Вариант 1 — внутри tmux
Ctrl+A затем Shift+I    # установить плагины

# Вариант 2 — из терминала
~/.tmux/plugins/tpm/bin/install_plugins
```

---

## Запуск

```bash
tmux                      # новая сессия
tmux new -s work          # новая сессия с именем
tmux ls                   # список сессий
tmux attach -t work       # подключиться к сессии
tmux kill-session -t work # удалить сессию
tmux kill-server          # убить все сессии
```

---

## Сессии

| Клавиша | Действие |
|---------|----------|
| `Ctrl+A s` | выбрать сессию |
| `Ctrl+A $` | переименовать сессию |
| `Ctrl+A d` | отключиться (сессия остаётся) |

---

## Окна (вкладки)

| Клавиша | Действие |
|---------|----------|
| `Ctrl+A c` | новое окно (в текущей папке) |
| `Ctrl+A ,` | переименовать окно |
| `Ctrl+A n` | следующее окно |
| `Ctrl+A p` | предыдущее окно |
| `Ctrl+A 1-9` | перейти к окну по номеру |
| `Ctrl+A &` | закрыть окно |

---

## Панели (splits)

| Клавиша | Действие |
|---------|----------|
| `Ctrl+A \|` | разделить вертикально |
| `Ctrl+A -` | разделить горизонтально |
| `Ctrl+A h/j/k/l` | навигация между панелями |
| `Ctrl+A H/J/K/L` | изменить размер панели |
| `Ctrl+A z` | развернуть панель на весь экран |
| `Ctrl+A x` | закрыть панель |

---

## Просмотр доки в сплите

```bash
# 1. Открываем tmux
tmux

# 2. Делаем вертикальный split
Ctrl+A |

# 3. В новой панели открываем доку
glow ~/dotfiles/docs/tmux.md

# 4. Переключаемся между панелями
Ctrl+A h/l
```

---

## Копирование текста

| Клавиша | Действие |
|---------|----------|
| `Ctrl+A Enter` | войти в режим копирования |
| `v` | начать выделение |
| `y` | скопировать |
| `Escape` | выйти |

> Чтобы выделить мышкой — зажми `Shift` и выделяй как обычно.

## Сохранение сессий (tmux-resurrect)

| Клавиша | Действие |
|---------|----------|
| `Ctrl+A Ctrl+S` | сохранить сессию |
| `Ctrl+A Ctrl+R` | восстановить сессию |

> tmux-continuum сохраняет автоматически каждые 15 минут.

---

## SSH workflow

```bash
# Подключаемся по ssh
ssh user@server

# На сервере создаём сессию
tmux new -s main

# Если соединение оборвалось — переподключаемся
ssh user@server
tmux attach -t main
```

> Сессия остаётся живой на сервере даже если ssh соединение оборвалось.

## Сохранение сессий (tmux-resurrect)

| Клавиша | Действие |
|---------|----------|
| `Ctrl+A Ctrl+S` | сохранить текущее состояние |
| `Ctrl+A Ctrl+R` | восстановить сессии |

> tmux-continuum сохраняет автоматически каждые 15 минут.

### Как правильно закрыть сессию навсегда

```bash
# 1. Удалить сессию
tmux kill-session -t имя

# 2. Сразу сохранить чистое состояние
Ctrl+A Ctrl+S

# Иначе continuum восстановит удалённую сессию при следующем запуске
```

### Сбросить все сохранённые сессии

```bash
rm -rf ~/.local/share/tmux/resurrect

tmux kill-server
```
