# Tmux — Шпаргалка

Терминальный мультиплексор — несколько терминалов в одном окне,
сессии живут даже после закрытия терминала.

Префикс: `Ctrl+B` (стандартный)
Все команды: сначала префикс, потом клавиша.

---

## Установка плагинов

После первого запуска или добавления плагина в конфиг:

```bash
# Вариант 1 — внутри tmux
Ctrl+B затем I    # установить плагины

# Вариант 2 — из терминала
~/.tmux/plugins/tpm/bin/install_plugins
```

---

## Запуск

```bash
tmux                      # новая сессия
tmux new -s work          # новая сессия с именем
tmux ls                   # список сессий
tmux attach -t work       # подключиться к сессии
tmux kill-session -t work # удалить сессию
tmux kill-server          # убить все сессии
```

---

## Сессии

| Клавиша | Действие |
|---------|----------|
| `Ctrl+B s` | выбрать сессию |
| `Ctrl+B $` | переименовать сессию |
| `Ctrl+B d` | отключиться (сессия остаётся) |

---

## Окна (вкладки)

| Клавиша | Действие |
|---------|----------|
| `Ctrl+B c` | новое окно (в текущей папке) |
| `Ctrl+B ,` | переименовать окно |
| `Ctrl+B n` | следующее окно |
| `Ctrl+B p` | предыдущее окно |
| `Ctrl+B 1-9` | перейти к окну по номеру |
| `Ctrl+B &` | закрыть окно |

---

## Панели (splits)

| Клавиша | Действие |
|---------|----------|
| `Ctrl+B \|` | разделить вертикально |
| `Ctrl+B -` | разделить горизонтально |
| `Ctrl+B h/j/k/l` | навигация между панелями |
| `Ctrl+B H/J/K/L` | изменить размер панели |
| `Ctrl+B z` | развернуть панель на весь экран |
| `Ctrl+B x` | закрыть панель |

---

## Просмотр доки в сплите

```bash
# 1. Открываем tmux
tmux

# 2. Делаем вертикальный split
Ctrl+B |

# 3. В новой панели открываем доку
glow ~/dotfiles/docs/tmux.md

# 4. Переключаемся между панелями
Ctrl+B h/l
```

---

## Копирование текста

| Клавиша | Действие |
|---------|----------|
| `Ctrl+B Enter` | войти в режим копирования |
| `v` | начать выделение |
| `y` | скопировать |
| `Escape` | выйти |

> Чтобы выделить мышкой — зажми `Shift` и выделяй как обычно.

---

## Сохранение сессий (tmux-resurrect)

| Клавиша | Действие |
|---------|----------|
| `Ctrl+B Ctrl+S` | сохранить сессию |
| `Ctrl+B Ctrl+R` | восстановить сессию |

> tmux-continuum сохраняет автоматически каждые 15 минут.

---

## SSH workflow

```bash
# Подключаемся по ssh
ssh user@server

# На сервере создаём сессию
tmux new -s main

# Если соединение оборвалось — переподключаемся
ssh user@server
tmux attach -t main
```

> Сессия остаётся живой на сервере даже если ssh соединение оборвалось.
